# Face K&B Classifier

Система автоматической классификации лиц для определения принадлежности к сотрудникам сети "КБ".

## Основной функционал

- **CNN классификация:** Определение по фотографии (Работники КБ / Не работники КБ)
- **Маркировка:** Автоматическое наложение водяного знака по диагонали изображения
- **Грейскейл-фильтр:** Преобразование изображения в оттенки серого
- **Отчетность:** Генерация HTML-отчетов и PowerPoint-презентаций

## Архитектура модели
Input (3, 128, 128)

Conv2d(3→32) + BatchNorm + ReLU + MaxPool
Conv2d(32→64) + BatchNorm + ReLU + MaxPool
Conv2d(64→128) + BatchNorm + ReLU + MaxPool
Conv2d(128→256) + BatchNorm + ReLU + MaxPool
Flatten (256 * 8 * 8 = 16384)
 Linear(16384→512) + ReLU + Dropout(0.5)
Linear(512→128) + ReLU + Dropout(0.5)
Linear(128→2) → Output (2 classes)

**Параметры модели:** ~8.5M обучаемых параметров
